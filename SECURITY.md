# Security policy

CFBoundary is a small FFI boundary library. Please report security issues privately rather than opening a public issue.

## What to report

Examples:

- boundary conversions that can leak secrets or bypass validation;
- unsafe default behavior for Cloudflare bindings;
- cache, service-binding, or deploy helpers that could expose private data;
- vulnerabilities in examples or documentation that users are likely to copy.

## Supported versions

Before `1.0`, only the latest commit/tag is supported.
